Repository List

This is a place for us to mirror github repositories. You cannot git push here; it's a read-only mirror away from github. Your repo will be cloned from github whenever github sends gitmirror a web hook request. The URLs will mirror the /:username/:repo structure from github.

You can git clone from this page. For example, github.com/jbalogh/reflecto can be cloned with git clone git://gitmirror.mozilla.org/jbalogh/reflecto.

Setup instructions are on the intranet.
